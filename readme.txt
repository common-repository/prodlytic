Seamlessly connect Prodlytic, the analytics tool that automatically records every user interaction from day one, and start understanding what people do on your website straight away.
 
== Description ==
 
Prodlytic is analytics that records every user interaction from day one so you can focus on the metrics that matter right now without sacrificing data for the future.

You add, change, or remove metrics as if you’re using your product or browsing your website, and historical data populates your dashboard straight away. Automatic Slack updates or email reports make sure you always know the important data, creating constant context for building a product.

* Never miss data again with always on recording
* Visually add events, no code
* Create in-depth funnels and features, helping you understand your users
* Group users in behavioural cohorts to find trends

If you don’t already have a free Prodlytic account you can sign up here: [prodlytic.com](https://prodlytic.com/)
 
== Installation ==
 
1. Install the plugin from the WordPress directory
1. Click Prodlytic in the left hand admin menu
1. Enter in your product ID into the settings page

Prodlytic is now configured and will be collecting analytics data.
